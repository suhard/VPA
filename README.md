# VPA
Basic K8s Vertical Pod Autoscaling example. For reference, please check:
https://github.com/kubernetes/autoscaler/blob/master/vertical-pod-autoscaler/docs/quickstart.md

VPA installation steps in the K8s cluster (further details: https://oneuptime.com/blog/post/2026-01-06-kubernetes-vpa-vertical-pod-autoscaling/view):
git clone https://github.com/kubernetes/autoscaler.git
cd autoscaler/vertical-pod-autoscaler
./hack/vpa-up.sh
